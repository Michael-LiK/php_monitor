## Installing

```shell
$ composer require mini-monitor/php_monitor
```

## Usage

There are three functions you can use .

example:

Add one to on the key
```shell 
add($key) 
```  
To add a value of key
```shell 
addValue($key,$value)
```

To set a value of key
```shell 
set($key,$value)  
```
